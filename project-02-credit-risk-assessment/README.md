# 🚀 Project 02 – Credit Risk Assessment

Enterprise Machine Learning project for predicting customer credit risk using financial analytics and classification models.

---

# Overview

This project predicts whether a loan applicant is considered low risk or high risk based on financial and demographic data.

The solution helps financial institutions:

- Improve loan approval decisions
- Reduce default risk
- Automate credit assessment
- Support risk management teams
- Improve portfolio quality

---

# Business Problem

Banks and lenders need reliable models to assess credit risk before approving loans.

Manual assessment can be slow and inconsistent.

Machine Learning can help classify applicants based on risk probability.
