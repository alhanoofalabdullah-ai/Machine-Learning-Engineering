# 🚀 Project 10 – Resume Screening AI

Enterprise AI-powered Resume Screening Platform for candidate ranking, skills extraction, ATS scoring, and recruitment analytics.

---

# Overview

Recruiters receive hundreds of resumes for a single job opening.

This project automates resume screening using NLP and Machine Learning.

The platform helps organizations:

- Reduce screening time
- Improve hiring efficiency
- Rank candidates automatically
- Extract critical skills
- Improve ATS processes
- Support talent acquisition teams

---

# Business Problem

Manual resume review is time-consuming and inconsistent.

Organizations need intelligent systems capable of identifying the best candidates based on job requirements.

Machine Learning and NLP provide automated candidate evaluation.
